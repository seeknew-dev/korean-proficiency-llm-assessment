# 2. 연구개발 방법

## 2.1 연구개발 추진 전략

본 과제는 **5단계 접근법**으로 진행됩니다:

**1단계**: 데이터 수집 및 분석 (1-4개월)
**2단계**: 난이도 측정 모델 개발 (4-8개월)
**3단계**: 직군별 평가 시스템 개발 (8-12개월)
**4단계**: 학습자 맞춤형 추천 시스템 개발 (12-15개월)
**5단계**: 통합 검증 및 최적화 (15-18개월)

## 2.2 세부 연구개발 방법

### 2.2.1 1단계: 데이터 수집 및 분석 (1-4개월)

**목표**: 문제 난이도 측정을 위한 실제 수험자 응답 데이터 확보

**방법**:

1. **기존 EPS-TOPIK 문제 활용**
   - 자체 개발한 문제 생성 파이프라인으로 생성된 1,000개 이상의 문제 활용
   - 2026년 2월 오픈 예정인 서비스를 통해 실제 학습자 응답 데이터 수집

2. **데이터 수집 체계 구축**
   - PostgreSQL 기반 응답 데이터 저장소 구축
   - 수험자 ID, 문제 ID, 응답 시간, 정답 여부, 선택 선택지 등 수집
   - 개인정보 비식별화 처리

3. **응답 패턴 분석**
   - 문제별 정답률, 응답 시간 분포 분석
   - 수험자 수준별(초급/중급/고급) 정답률 차이 분석
   - 오답 선택지별 선택 비율 분석 (distractor 효과성 측정)

**검증 방법**:

- 최소 500명 이상의 학습자로부터 50,000개 이상의 응답 데이터 수집
- 데이터 품질 검증: 무작위 응답 패턴 제거, 비정상적 응답 시간 필터링

### 2.2.2 2단계: 난이도 측정 모델 개발 (4-8개월)

**목표**: 문제의 다차원 난이도를 자동으로 측정하는 AI 모델 개발

**방법**:

1. **다차원 난이도 정의**
   - 어휘 난이도: 사용된 단어의 빈도, 급수 (1급~6급)
   - 문법 복잡도: 문장 길이, 복문 사용, 고급 문법 요소
   - 독해 수준: 지문의 추상성, 함축적 의미 포함 여부
   - 논리적 사고: 추론, 비교, 인과관계 파악 필요 정도
   - 실용적 기능: 실생활 맥락 이해도 (격식체/비격식체, 상황 판단 등)

2. **IRT(Item Response Theory) 기반 분석**
   - 1단계에서 수집한 응답 데이터를 IRT 모델에 적용
   - 문항 난이도(difficulty), 변별도(discrimination), 추측도(guessing) 계산
   - Python `mirt` 또는 `pyirt` 라이브러리 활용

3. **LLM 기반 난이도 예측 모델**
   - LLM을 활용하여 문제 텍스트로부터 난이도 요소 추출
   - Few-shot learning: IRT로 측정된 실제 난이도를 학습 데이터로 활용
   - Fine-tuning: 한국어 문제 특화 난이도 예측 모델 학습
   - Vertex AI Pipeline을 활용한 자동 학습 파이프라인 구축

4. **오답 선택지 품질 평가**
   - Distractor 매력도 측정: 각 오답 선택지가 실제로 선택된 비율 분석
   - 너무 쉽게 배제되는 선택지 자동 탐지 및 개선 제안
   - 오답 선택지의 교육학적 타당성 평가 (흔한 오류 패턴 반영 여부)

**검증 방법**:

- 예측 난이도와 실제 정답률 간 상관계수 0.85 이상
- 교차 검증: 신규 문제의 예측 정답률과 실제 정답률 비교
- 전문가 평가: 한국어교육 전문가 3인의 주관적 난이도 평가와 비교

### 2.2.3 3단계: 직군별 전문 언어 평가 시스템 개발 (8-12개월)

**목표**: 공통 영역과 직군별 영역을 분리한 맞춤형 평가 문제 자동 생성

**방법**:

1. **공통 영역 및 직군별 영역 분리 설계**
   - **공통 영역 (일상생활)**: 모든 직군에 공통으로 필요한 언어 능력
     - 쇼핑, 병원, 은행, 관공서, 교통, 이웃 관계 등
   - **직군별 영역**: 각 직군에 특화된 전문 용어 및 상황별 표현
     - 건축 시공: 안전 용어(안전모, 비계, 추락 방지), 공구명, 작업 지시
     - 토목 시공: 중장비 명칭, 토목 용어, 공사 안전
     - 기계 조립: 공정 용어(선반, 용접, 조립), 기계 부품명
     - 금속 가공: 가공 기법, 측정 도구, 품질 관리
     - 음식 조리: 조리 기법, 위생 관리, 주문 처리, 식재료
     - 숙박 서비스: 객실 관리, 체크인/아웃, 고객 응대
     - 작업장 관리: 재고 관리, 작업 지시, 일정 조율

2. **직군별 측정 요소 정의**
   - 각 직군에서 평가해야 할 핵심 언어 능력 요소 정의
   - 예: 건축 시공 → 안전 용어 이해, 명령형 표현 이해, 위험 상황 판단, 공구 요청 표현
   - 예: 기계 조립 → 공정 용어 이해, 작업 지시 이해, 불량 보고 표현, 도면 관련 용어
   - 예: 숙박 서비스 → 손님 응대 표현, 예약 관련 용어, 시설 안내 능력, 클레임 처리

3. **직군별 맞춤형 문제 생성 알고리즘**
   - LLM에 직군별 전문 지식을 프롬프트로 주입
   - 직군별 필수 어휘, 문법, 상황을 포함한 문제 자동 생성
   - 지정된 난이도 벡터에 맞춰 문제 난이도 조정
   - 예: "건축 시공, 중급, 안전 용어 중심, 명령형 표현 이해" → 해당 조건에 맞는 문제 생성
   - 예: "숙박 서비스, 초급, 객실 관리 용어, 고객 응대 표현" → 해당 조건에 맞는 문제 생성

4. **평가 문제 세트 구성 알고리즘**
   - **공통 영역 문제** (일상생활 한국어) + **직군별 문제** (전문 용어/상황) 자동 조합
   - 목표 점수 분포를 입력하면 이에 맞는 문제 조합 자동 선정
   - 직군 내 다양한 하위 주제를 균형 있게 포함 (안전, 작업, 의사소통 등)
   - 문법/어휘/독해/실용 영역의 비율 조정 가능

5. **세분화된 평가 결과 산출**
   - "일상생활 점수" + "직군별 점수" 분리 제공
   - 기업이 지원자의 "자사 직군 관련 언어 능력"을 명확히 파악 가능

**검증 방법**:

- 직군별 전문가 검증: 건축 시공 종사자, 기계 조립 종사자, 음식 조리 종사자, 숙박 서비스 종사자 등의 실제 피드백 수집
- 문제의 직군 적합성 평가: 각 직군 전문가 3인의 블라인드 평가 (정확도 90% 이상)
- A/B 테스트: 직군별 맞춤 문제와 일반 직장생활 문제의 예측력 비교
- 점수 분리 효과 검증: "직군별 점수"가 실제 업무 수행 능력과의 상관관계가 "총점"보다 높은지 측정

### 2.2.4 4단계: 학습자 맞춤형 추천 시스템 개발 (12-15개월)

**목표**: 학습자 맞춤형 문제 추천 시스템 개발

**방법**:

1. **학습자 수준 진단 및 분석 시스템**
   - **현재 수준 파악**: 평가 결과 기반 학습자의 다차원 능력 프로파일 생성
     - 어휘 수준, 문법 이해도, 독해 능력, 직군별 전문 용어 숙지도 등
   - **약점 영역 식별**: 다차원 난이도 측정 데이터로 학습자의 취약 영역 자동 탐지
     - 예: "건축 시공 지망자 - 안전 용어는 우수하나 작업 지시 이해도 낮음"
   - **학습 진도 추적**: 시간에 따른 능력 향상 추이 분석

2. **맞춤형 문제 추천 알고리즘**
   - **약점 보완 우선**: 학습자의 취약 영역(어휘/문법/독해/직군)에 해당하는 문제 우선 제공
   - **적정 난이도 자동 조정**: 학습자 수준보다 약간 높은 난이도의 문제 제공 (Zone of Proximal Development)
     - 정답률 70~80% 수준의 문제로 구성하여 성취감 유지하면서 도전 제공
   - **단계적 난이도 상승**: 학습자의 정답률이 일정 수준 이상 유지되면 자동으로 난이도 상승
   - **학습 경로 최적화**: 학습자의 목표(직군, 목표 점수)에 따른 최적 학습 순서 제시
     - 예: "기초 일상생활 어휘 → 직군 기본 용어 → 직군 고급 표현" 순서

3. **동형 문제 대량 생성 시스템**
   - 동일 난이도/변별력을 가진 문제를 무한 생성하는 알고리즘 개발
   - Paraphrasing 기반: 표면적 표현은 다르지만 측정 능력은 동일한 문제 생성
   - 어휘/구문 변형을 통한 변형 생성 (예: "안전모를 착용하세요" → "작업 시 머리 보호구를 반드시 쓰세요")
   - 문제 풀 관리 시스템: 난이도/영역/유형별 문제 DB 구축, 노출 이력 관리

**검증 방법**:

- **추천 정확도**: 학습자가 추천받은 문제의 정답률이 70~80% 범위 내 유지되는 비율 (목표: 80% 이상)
- 동형 문제 간 난이도 일치도: 평균 점수 편차 ±5% 이내

### 2.2.5 5단계: 통합 검증 및 최적화 (15-18개월)

**목표**: 전체 시스템 통합 및 실전 검증 (단기 + 장기 학습 효과 검증)

**방법**:

1. **통합 평가 및 학습 플랫폼 구축**
   - React Native (Expo) 기반 크로스 플랫폼 앱에 통합
   - **평가 모드**: 직군별 세분화된 평가 제공
   - **학습 모드**: 맞춤형 문제 추천 및 학습 진도 관리
   - GCP Cloud Run을 통한 백엔드 API 배포
   - Vertex AI Pipeline을 통한 문제 생성 자동화

2. **단기 학습 효과 검증** (4주)
   - 베타 테스터 100명 이상 모집하여 실제 평가 진행
   - 4주간 맞춤형 학습 후 능력 향상 측정 (목표: 평균 20% 이상 점수 향상)
   - 외국인 근로자 채용 희망 기업 3곳 이상과 파일럿 테스트 진행
   - 기존 TOPIK/EPS-TOPIK 점수와의 상관관계 분석

3. **장기 학습 효과 검증** (3개월) - 자사 서비스 기반
   - **자사 한국어 교육 앱 활용**: 2026년 2월 오픈 예정인 서비스
   - **실험군**: 맞춤형 추천 시스템 적용 학습자 50명 이상
   - **대조군**: 일반 난이도 순차 학습 학습자 50명 이상
   - **측정 기간**: 16~18개월차 (3개월간 지속 관찰)
   - **측정 항목**:
     - 월별 모의 평가 점수 변화 추이
     - 학습 지속률 (이탈률) 비교
     - 문제 정답률 향상 속도
     - 약점 영역 개선도
   - **최종 평가**: 3개월 후 표준화된 한국어 능력 평가 실시

4. **성능 최적화**
   - 문제 생성 속도 향상: 평균 1초 이내 문제 생성
   - 비용 절감: 효율적인 LLM 활용으로 API 비용 최소화
   - 확장성 확보: 동시 접속자 1,000명 이상 처리 가능

**검증 방법**:

- **단기 학습 효과**: 4주간 학습 후 평균 점수 향상률 20% 이상
- **장기 학습 효과**:
  - 실험군의 평균 점수 향상률이 대조군 대비 20% 이상 높음
  - 실험군의 학습 지속률이 대조군 대비 30% 이상 높음
  - 실험군의 약점 영역 개선 속도가 대조군 대비 유의미하게 빠름 (p < 0.05)
- 실제 언어 능력 예측력: 기업의 실제 업무 수행 능력 평가와 상관계수 0.8 이상
- 시스템 안정성: 99.9% 가용성 달성
- 비용 효율성: 기존 대비 70% 비용 절감

## 2.3 기술적 차별성

| 기존 기술 (EPS-TOPIK)                                  | 본 과제의 차별성                                                                     |
| ------------------------------------------------------ | ------------------------------------------------------------------------------------ |
| 단일 난이도 지표 (쉬움/보통/어려움)                    | 다차원 벡터 (어휘, 문법, 독해, 논리, 기능 등 5개 이상 차원)                          |
| 전문가 주관적 평가로 난이도 결정                       | 실제 수험자 데이터 기반 객관적 측정 (IRT + LLM)                                      |
| 모든 직군 혼합 출제, 총점만 제공                       | **공통 영역 + 직군별 영역 분리 평가**, 세분화된 점수 제공 (일상생활 + 직군별 점수) |
| 숙박 지망생도 건설 용어, 건설 지망생도 서비스 용어 학습 | 희망 직군에 맞춘 선택적 평가로 학습 부담 최소화                                      |
| **평가 기능만 제공** (학습 지원 없음)                  | **평가 + 맞춤형 학습 통합**: 약점 분석 기반 문제 추천, 단계적 난이도 조정            |
| 제한된 문제 풀 (재사용으로 유출 위험)                  | 동형 문제 무한 생성 (부정행위 방지)                                                  |
| 문제 제작 비용 문항당 5만원 이상                       | 자동 생성으로 문항당 1천원 이하                                                      |

## 2.4 R&D 결과 검증 방법 및 계획

### 검증 방법 1: 정량적 지표

- **난이도 예측 정확도**: 신규 문제 100개에 대해 예측 정답률과 실제 정답률 비교 (목표: 상관계수 0.85 이상)
- **문항 변별도**: IRT 분석을 통해 변별도 0.4 이상인 문제 비율 측정 (목표: 80% 이상)
- **동형 문제 일치도**: 동일 조건의 문제 세트 10개 생성 후 평균 점수 편차 측정 (목표: ±5% 이내)

### 검증 방법 2: 전문가 평가

- 한국어교육 전문가 3인 이상의 블라인드 평가
- 문제의 교육학적 타당성, 범주 적합성, 오답 선택지 품질 평가
- 평가 항목별 5점 척도 평가 (목표: 평균 4.0 이상)

### 검증 방법 3: 실전 테스트

- 베타 테스터 100명 이상 모집하여 실제 평가 진행
- 외국인 근로자 채용 희망 기업 3곳 이상과 파일럿 테스트
- 평가 결과와 실제 업무 수행 능력 간 상관관계 분석 (목표: 0.8 이상)

### 검증 방법 4: 자사 서비스 기반 장기 학습 효과 검증

**목표**: 개발된 맞춤형 추천 시스템의 실제 학습 효과를 자사 한국어 교육 앱을 통해 장기간 검증

**방법**:

1. **자사 서비스 활용**
   - 2026년 2월 오픈 예정인 자사 한국어 교육 앱 활용
   - 연구 기간(18개월) 동안 실제 학습자 데이터 지속 수집 및 분석
   - 학습자 동의 하에 학습 패턴, 진도, 성취도 데이터 수집

2. **학습자 그룹 구성**
   - **실험군**: 맞춤형 추천 시스템을 적용한 학습자 그룹 (50명 이상)
   - **대조군**: 일반 난이도 순차 학습을 진행하는 학습자 그룹 (50명 이상)
   - 초기 한국어 능력 수준을 동일하게 매칭하여 공정한 비교 수행

3. **장기 관찰 및 측정**
   - **관찰 기간**: 3개월 이상 지속 학습
   - **측정 지표**:
     - 월별 모의 평가 점수 변화 추이
     - 학습 지속률 (이탈률 비교)
     - 문제 정답률 향상 속도
     - 약점 영역 개선도
   - **최종 평가**: 3개월 후 표준화된 한국어 능력 평가 실시

4. **검증 기준**
   - 실험군의 평균 점수 향상률이 대조군 대비 **20% 이상 높음**
   - 실험군의 학습 지속률이 대조군 대비 **30% 이상 높음**
   - 실험군의 약점 영역 개선 속도가 대조군 대비 **유의미하게 빠름** (통계적 유의수준 p < 0.05)

5. **추가 분석**
   - 학습자 만족도 설문 조사 (5점 척도, 목표: 평균 4.0 이상)
   - 학습 시간 대비 효율성 분석 (동일 성취도 도달까지 소요 시간 비교)
   - 직군별 맞춤 학습 효과 비교 (건축, 기계, 음식, 숙박 등)

**기대 효과**:

- 실제 사용 환경에서의 장기간 검증을 통한 시스템 신뢰성 확보
- 자사 서비스 데이터 축적을 통한 지속적인 알고리즘 개선
- 연구 종료 후 즉시 상용화 가능한 검증된 서비스 확보
