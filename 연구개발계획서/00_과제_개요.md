# 0. 과제 개요

## 과제명

**한국어 능력 평가를 위한 LLM 기반 다차원 난이도 측정 및 맞춤형 문제 생성 시스템 개발**

## 개발 목표

- 객관식 문제의 다차원 난이도 자동 측정 시스템 개발 (어휘, 문법, 독해, 논리, 기능 등)
- 수험자 응답 패턴 기반 문항 변별력 최적화 알고리즘 개발
- **직군별 전문 언어 평가 시스템** 개발 (공통 영역 + 직군별 영역 분리 평가)
- 일정 수준 이상의 한국어 능력 보유자 선발을 위한 고변별력 문제 생성 기술 개발

## 개발 방법

- LLM 기반 문제 생성 파이프라인 고도화
- 실제 수험자 응답 데이터 수집 및 분석을 통한 난이도 예측 모델 학습
- IRT(Item Response Theory) 기반 문항 변별도, 난이도 자동 평가 시스템 구축
- **직군별 필수 언어 능력 측정 요소 정의** 및 문제 자동 생성 알고리즘 개발
  - 공통 영역 (일상생활) + 직군별 영역 (전문 용어/상황) 분리 생성
- 동형 문제 대량 생성을 통한 부정행위 방지 시스템 구축
