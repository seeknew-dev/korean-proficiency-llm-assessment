# 0. 과제 개요

## 과제명

**외국인 근로자 대상 기업 맞춤형 한국어 능력 평가 및 학습 지원 시스템 개발**

## 개발 목표

- 객관식 문제의 다차원 난이도 자동 측정 시스템 개발 (어휘, 문법, 독해, 논리, 기능 등)
- 수험자 응답 패턴 기반 문항 변별력 최적화 알고리즘 개발
- **직군별 전문 언어 평가 시스템** 개발 (공통 영역 + 직군별 영역 분리 평가)
- **학습자 맞춤형 추천 시스템** 개발 (약점 분석 및 적정 난이도 문제 추천)

## 개발 방법

- 자사 한국어 교육 서비스를 활용한 실제 수험자 응답 데이터 수집 및 분석 (50,000건 이상)
- IRT(Item Response Theory) 기반 문항 변별도, 난이도 자동 평가 시스템 구축
- **클라우드 GPU에서 LLM Fine-tuning 후 사내 서버 배포**
  - 7B~8B 파라미터급 오픈소스 LLM 활용
  - 4-bit 양자화를 통한 사내 추론 서버 구축 (8개월차)
- **직군별 필수 언어 능력 측정 요소 정의** 및 문제 자동 생성 알고리즘 개발
  - 공통 영역 (일상생활) + 직군별 영역 (전문 용어/상황) 분리 생성
- 학습자 약점 분석 및 맞춤형 문제 추천 알고리즘 개발
- 동형 문제 대량 생성을 통한 부정행위 방지 시스템 구축

## 연구팀 구성

**현재 인력** (총 3명, 여성 2명, 남성 1명, 여성 비율 66.7%)
- 연구책임자 겸 AI/NLP 연구원: [직접 작성] (여성/남성)
- 데이터 분석 및 UX 연구원: [직접 작성] (여성/남성)
- 백엔드/앱 개발자: [직접 작성] (여성/남성)

**과제 수행 중 채용 예정** (총 2명, 여성 우선 채용)
- 한국어교육 전문가: 1명 (1분기 채용 예정, 여성 우선)
- 백엔드/인프라 개발자: 1명 (상반기 채용 예정, 여성 우선)

**채용 후 팀 구성**: 총 5명 (목표: 여성 비율 60% 이상 유지)

**외부 전문기술 활용**: 한국어교육 전문가 자문 (직군별 평가 시스템 개발 및 품질 검증)

## 기대효과

- EPS-TOPIK, TOPIK, 세종학당 등 한국어 능력 시험 기관에 기술 제공
- 외국인 근로자 채용 기업의 한국어 능력 검증 비용 70% 절감
- 기업별 맞춤형 한국어 능력 평가 서비스 시장 진출
- 연 매출 10억원 달성 및 일자리 4개 창출 (3년 내)
- **여성 고용 확대**: 여성 비율 60% 이상 유지, AI 분야 여성 전문 인력 양성
