# 5. 기술개발일정

## 전체 일정 (1년 6개월, 18개월)

**연구개발 기간**: 2026.4.1 ~ 2027.9.30 (18개월)
- 1차년도: 2026.4.1 ~ 2026.12.31 (9개월)
- 2차년도: 2027.1.1 ~ 2027.9.30 (9개월)

| 단계  | 기간      | 주요 내용                      | 산출물                                     |
| ----- | --------- | ------------------------------ | ------------------------------------------ |
| 1단계 | 1~4개월   | 데이터 수집 및 분석            | 응답 데이터 50,000건 이상, 분석 보고서     |
| 2단계 | 4~8개월   | 난이도 측정 모델 개발          | IRT 기반 난이도 측정 모델, 예측 정확도 90% |
| 3단계 | 8~12개월  | 직군별 평가 시스템 개발        | 직군별 문제 생성 엔진, 직군 DB             |
| 4단계 | 12~15개월 | 학습자 맞춤형 추천 시스템 개발 | 맞춤형 추천 알고리즘, 동형 문제 생성 엔진  |
| 5단계 | 15~18개월 | 통합 검증 및 최적화            | 통합 플랫폼, 실전 테스트 보고서            |

## 세부 일정

### 1~4개월: 1단계 - 데이터 수집 및 분석 (1차년도)

- **1~2개월차** (2026.4~5월)
  - 서비스 정식 오픈 및 초기 사용자 유치
  - 응답 데이터 수집 체계 구축 (PostgreSQL 스키마 설계)
  - 개인정보 비식별화 프로세스 구축
  - 기초 통계 분석: 문제별 정답률, 응답 시간 분포

- **3~4개월차** (2026.6~7월)
  - 누적 응답 데이터 50,000건 이상 수집
  - 이상 데이터 필터링 (무작위 응답, 비정상적 응답 시간 등)
  - 수험자 수준별 정답률 차이 분석
  - 오답 선택지별 매력도 분석
  - **산출물**: 응답 데이터 분석 보고서

### 4~8개월: 2단계 - 난이도 측정 모델 개발 (1차년도)

- **5개월차** (2026.8월)
  - IRT(Item Response Theory) 모델 적용
  - 문항 난이도, 변별도, 추측도 계산
  - 다차원 난이도 정의 및 지표 설계

- **6~7개월차** (2026.9~10월)
  - LLM 기반 난이도 예측 모델 개발
  - Few-shot learning 프롬프트 설계
  - Vertex AI Pipeline 자동 학습 파이프라인 구축

- **8개월차** (2026.11월)
  - 모델 성능 평가 및 개선
  - 오답 선택지 품질 평가 알고리즘 개발
  - **산출물**: 난이도 측정 모델 (예측 정확도 90% 이상)
  - **중간 점검**: 난이도 예측 정확도 측정

### 8~12개월: 3단계 - 직군별 평가 시스템 개발 (1차년도 말 ~ 2차년도 초)

- **9개월차** (2026.12월, **1차년도 종료**)
  - 직군 정의 및 분류 체계 구축 (공통 영역 + 직군별 영역: 건축 시공/기계 조립/음식 조리/숙박 서비스 등)
  - 직군별 필수 어휘 리스트 구축 착수 (직군당 1,000개 이상)

- **10~11개월차** (2027.1~2월, **2차년도 시작**)
  - 직군별 필수 어휘 리스트 구축 완료
  - 한국어교육 전문가 자문 회의 (5회)
  - 직군별 측정 요소 정의 (전문 용어, 작업 지시 이해, 상황별 표현 등)
  - 맞춤형 문제 생성 알고리즘 개발

- **12개월차** (2027.3월)
  - 직군별 프롬프트 템플릿 설계
  - 문제 세트 자동 구성 알고리즘 개발
  - 직군별 문제 생성 테스트 (각 직군당 100문항)
  - 전문가 검증 및 피드백 반영
  - **산출물**: 직군별 문제 생성 엔진, 직군 DB
  - **중간 검증**: 전문가 평가 (적합성 90% 이상)

### 12~15개월: 4단계 - 학습자 맞춤형 추천 시스템 개발 (2차년도)

- **13개월차** (2027.4월)
  - 학습자 수준 진단 및 분석 시스템 개발
  - 다차원 능력 프로파일 생성 알고리즘 구현

- **14개월차** (2027.5월)
  - 맞춤형 문제 추천 알고리즘 개발 (약점 보완 우선, 적정 난이도 자동 조정)
  - 학습 경로 최적화 알고리즘 개발

- **15개월차** (2027.6월)
  - 동형 문제 대량 생성 시스템 개발
  - Paraphrasing 알고리즘 구현
  - 문제 풀 관리 시스템 구축
  - **산출물**: 맞춤형 추천 알고리즘, 동형 문제 생성 엔진
  - **중간 검증**: 추천 정확도 측정 (정답률 70~80% 범위 유지 비율 80% 이상)

### 15~18개월: 5단계 - 통합 검증 및 최적화 (2차년도)

- **16개월차** (2027.7월)
  - 통합 평가 및 학습 플랫폼 구축 (React Native 앱 + Cloud Run API)
  - 평가 모드 + 학습 모드 통합
  - GCP Cloud Run 배포 및 Vertex AI Pipeline 자동화
  - **장기 학습 효과 검증 시작**: 자사 앱 기반 실험군/대조군 3개월 학습 프로그램 시작

- **16~18개월차** (2027.7~9월)
  - **장기 학습 효과 검증 진행** (3개월간 지속 관찰)
    - 실험군: 맞춤형 추천 시스템 적용 학습자 50명 이상
    - 대조군: 일반 난이도 순차 학습자 50명 이상
    - 월별 모의 평가 및 학습 패턴 데이터 수집
    - 학습 지속률, 점수 향상률, 약점 개선도 측정

- **17개월차** (2027.8월)
  - 베타 테스터 100명 모집 및 테스트 진행
  - 파일럿 기업 3곳과 실전 테스트
  - 단기 학습 효과 검증: 4주간 맞춤형 학습 후 능력 향상 측정

- **18개월차** (2027.9월, **2차년도 종료**)
  - **장기 학습 효과 최종 평가**: 3개월 학습 완료 후 표준화된 한국어 능력 평가 실시
  - 실험군 vs 대조군 학습 효과 비교 분석 (목표: 실험군 20% 이상 높은 향상률)
  - 성능 최적화 (속도, 비용, 확장성)
  - 최종 검증 및 보고서 작성
  - **산출물**: 통합 평가 및 학습 플랫폼, 장기 학습 효과 검증 보고서, 최종 연구 보고서
  - **최종 검증**:
    - 단기 학습 효과: 4주 평균 20% 이상 점수 향상
    - 장기 학습 효과: 3개월 실험군이 대조군 대비 20% 이상 높은 향상률
    - 학습 지속률: 실험군이 대조군 대비 30% 이상 높음
    - 전체 시스템 통합 테스트
