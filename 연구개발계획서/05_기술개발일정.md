# 5. 기술개발일정

## 전체 일정 (1년 6개월, 18개월)

**연구개발 기간**: 2026.4.1 ~ 2027.9.30 (18개월)
- 1차년도: 2026.4.1 ~ 2026.12.31 (9개월)
- 2차년도: 2027.1.1 ~ 2027.9.30 (9개월)

| 단계  | 기간       | 주요 내용                           | 산출물                                          |
| ----- | ---------- | ----------------------------------- | ----------------------------------------------- |
| 1단계 | 1~5개월    | 데이터 수집 및 분석                 | 응답 데이터 50,000건 이상, 분석 보고서          |
| 2단계 | 5~10개월   | 난이도 측정 모델 개발               | IRT 기반 난이도 측정 모델, 예측 정확도 90%      |
| 3단계 | 10~14개월  | 직군별 평가 시스템 개발             | 직군별 문제 생성 엔진, 직군 DB                  |
| 4단계 | 14~18개월  | 학습자 맞춤형 추천 시스템 및 검증  | 맞춤형 학습 플랫폼, 실전 테스트 보고서          |

## 세부 일정

### 1~5개월: 1단계 - 데이터 수집 및 분석 (1차년도)

- **1~2개월차** (2026.4~5월)
  - 서비스 정식 오픈 및 초기 사용자 유치
  - 응답 데이터 수집 체계 구축 (PostgreSQL 스키마 설계)
  - 개인정보 비식별화 프로세스 구축

- **3~4개월차** (2026.6~7월)
  - 누적 응답 데이터 20,000건 이상 수집
  - 기초 통계 분석: 문제별 정답률, 응답 시간 분포
  - 이상 데이터 필터링 (무작위 응답, 비정상적 응답 시간 등)

- **5개월차** (2026.8월)
  - 누적 응답 데이터 50,000건 이상 수집
  - 수험자 수준별 정답률 차이 분석
  - 오답 선택지별 매력도 분석
  - **산출물**: 응답 데이터 분석 보고서

### 5~10개월: 2단계 - 난이도 측정 모델 개발 (1차년도 말 ~ 2차년도 초)

- **6~7개월차** (2026.9~10월)
  - IRT(Item Response Theory) 모델 적용
  - 문항 난이도, 변별도, 추측도 계산
  - 다차원 난이도 정의 및 지표 설계

- **8~9개월차** (2026.11~12월, **1차년도 종료**)
  - LLM 기반 난이도 예측 모델 개발
  - Few-shot learning 프롬프트 설계
  - Vertex AI Pipeline 자동 학습 파이프라인 구축

- **10개월차** (2027.1월, **2차년도 시작**)
  - 모델 성능 평가 및 개선
  - 오답 선택지 품질 평가 알고리즘 개발
  - **산출물**: 난이도 측정 모델 (예측 정확도 90% 이상)
  - **중간 점검**: 난이도 예측 정확도 측정

### 10~14개월: 3단계 - 직군별 평가 시스템 개발 (2차년도)

- **11~12개월차** (2027.2~3월)
  - 직군 정의 및 분류 체계 구축 (공통 영역 + 직군별 영역: 건축 시공/기계 조립/음식 조리/숙박 서비스 등)
  - 직군별 필수 어휘 리스트 구축 (직군당 1,000개 이상)
  - 한국어교육 전문가 자문 회의 (5회)

- **13개월차** (2027.4월)
  - 직군별 측정 요소 정의 (전문 용어, 작업 지시 이해, 상황별 표현 등)
  - 맞춤형 문제 생성 알고리즘 개발
  - 직군별 프롬프트 템플릿 설계

- **14개월차** (2027.5월)
  - 문제 세트 자동 구성 알고리즘 개발
  - 직군별 문제 생성 테스트 (각 직군당 100문항)
  - 전문가 검증 및 피드백 반영
  - **산출물**: 직군별 문제 생성 엔진, 직군 DB
  - **중간 검증**: 전문가 평가 (적합성 90% 이상)

### 14~18개월: 4단계 - 학습자 맞춤형 추천 시스템 및 통합 검증 (2차년도)

- **15~16개월차** (2027.6~7월)
  - 학습자 수준 진단 및 분석 시스템 개발
  - 맞춤형 문제 추천 알고리즘 개발 (약점 보완 우선, 적정 난이도 자동 조정)
  - 동형 문제 대량 생성 시스템 개발
  - Paraphrasing 알고리즘 구현

- **17개월차** (2027.8월)
  - 통합 평가 및 학습 플랫폼 구축 (React Native 앱 + Cloud Run API)
  - 평가 모드 + 학습 모드 통합
  - 베타 테스터 100명 모집 및 테스트 진행
  - 파일럿 기업 3곳과 실전 테스트

- **18개월차** (2027.9월, **2차년도 종료**)
  - 학습 효과 검증: 4주간 맞춤형 학습 후 능력 향상 측정
  - 성능 최적화 (속도, 비용, 확장성)
  - 최종 검증 및 보고서 작성
  - **산출물**: 통합 평가 및 학습 플랫폼, 최종 연구 보고서
  - **최종 검증**: 학습 효과 검증 (평균 20% 이상 점수 향상), 전체 시스템 통합 테스트
