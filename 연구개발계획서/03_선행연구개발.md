# 3. 선행연구개발

**해당 사항 없음**

본 연구개발기관은 중기부 R&D 사업 또는 타 부처 지원 R&D 사업을 수행한 이력이 없습니다.

다만, **자체 연구개발 실적**은 다음과 같습니다:

## 자체 개발 실적

**과제명**: EPS-TOPIK 객관식 문제 자동 생성 시스템 개발
**기간**: 2025년 6월 ~ 2026년 1월 (8개월)
**투자 규모**: 자체 자금 5,000만원
**개발 내용**:

- Gemini 2.5 Flash 기반 문제 생성 파이프라인 구축
- EPS-TOPIK 수준의 다양한 형식의 문제 생성 가능 (독해, 문법, 어휘, 듣기 대본 등)
- PostgreSQL 기반 문제 데이터베이스 구축
- React Native (Expo) 기반 크로스 플랫폼 모바일 앱 개발
- GCP Cloud Run, Vertex AI Pipeline 인프라 구축

**현재 한계점**:

- 생성된 문제의 난이도 측정이 부정확 (전문가 주관 평가에 의존)
- 실제 수험자 응답 데이터 부족으로 문항 변별력 검증 불가
- 범용 문제만 생성 가능 (환경별 맞춤형 평가 불가)

**본 과제와의 차별성**:
본 과제는 자체 개발한 문제 생성 파이프라인을 기반으로, **난이도 측정의 정확성**, **범주별 맞춤형 평가**, **고변별력 문제 생성**이라는 핵심 기술을 새롭게 개발하는 것입니다. 기존 시스템은 "문제 생성"에 초점을 맞췄다면, 본 과제는 "정확한 능력 측정"과 "맞춤형 평가"라는 고부가가치 기술 개발을 목표로 합니다.
