# 5. 기술개발일정

## 연구개발 기간

**전체 기간**: 2026.4.1 ~ 2027.9.30 (18개월)
- 1차년도: 2026.4.1 ~ 2026.12.31 (9개월)
- 2차년도: 2027.1.1 ~ 2027.9.30 (9개월)

## 전체 개발 일정

| 단계  | 기간      | 주요 내용                      | 산출물                                     |
| ----- | --------- | ------------------------------ | ------------------------------------------ |
| 1단계 | 1~4개월   | 데이터 수집 및 분석            | 응답 데이터 50,000건 이상, 분석 보고서     |
| 2단계 | 4~8개월   | 난이도 측정 모델 개발          | IRT 기반 난이도 측정 모델, 예측 정확도 90% |
| 3단계 | 8~12개월  | 직군별 평가 시스템 개발        | 직군별 문제 생성 엔진, 직군 DB             |
| 4단계 | 12~15개월 | 학습자 맞춤형 추천 시스템 개발 | 맞춤형 추천 알고리즘, 동형 문제 생성 엔진  |
| 5단계 | 15~18개월 | 통합 검증 및 최적화            | 통합 플랫폼, 실전 테스트 보고서            |

## 단계별 세부 일정

### 1단계: 데이터 수집 및 분석 (1~4개월, 1차년도)

**배경**: 2026년 2월 자사 한국어 교육 서비스 정식 오픈, 4월부터 연구개발 착수

**1~2개월차** (2026.4~5월)
- 서비스 안정화 및 UX/UI 개선 (2월 오픈 후 피드백 반영)
- 응답 데이터 수집 체계 고도화 (PostgreSQL 스키마 확장, 비식별화 프로세스)
- 기초 통계 분석 대시보드 구축

**3~4개월차** (2026.6~7월)
- 누적 응답 데이터 50,000건 이상 수집 (2월~7월, 6개월간 누적)
- 이상 데이터 필터링 및 수험자 수준별 정답률 차이 분석
- **산출물**: 응답 데이터 분석 보고서

### 2단계: 난이도 측정 모델 개발 (4~8개월, 1차년도)

**5개월차** (2026.8월)
- IRT 모델 적용 및 문항 난이도·변별도·추측도 계산
- 다차원 난이도 정의 및 지표 설계

**6~7개월차** (2026.9~10월)
- 클라우드 GPU에서 LLM Fine-tuning (7B~8B 파라미터급, LoRA 기법)
- Few-shot learning 프롬프트 설계 및 AI 자동 학습 파이프라인 구축

**8개월차** (2026.11월)
- Fine-tuned 모델 4-bit 양자화 및 사내 추론 서버 배포
- 오답 선택지 품질 평가 알고리즘 개발
- **산출물**: 난이도 측정 모델 (예측 정확도 90% 이상), 사내 추론 서버

### 3단계: 직군별 평가 시스템 개발 (8~12개월, 1차년도 말~2차년도 초)

**9개월차** (2026.12월, 1차년도 종료)
- 직군 분류 체계 구축 (공통 영역 + 직군별 영역: 건축/기계/음식/숙박)
- 직군별 필수 어휘 리스트 구축 착수 (직군당 1,000개 이상)

**10~11개월차** (2027.1~2월, 2차년도 시작)
- 직군별 필수 어휘 리스트 완료 및 측정 요소 정의
- 맞춤형 문제 생성 알고리즘 개발

**12개월차** (2027.3월)
- 직군별 프롬프트 템플릿 설계 및 문제 세트 자동 구성
- 직군별 문제 생성 테스트 (각 직군당 100문항)
- **산출물**: 직군별 문제 생성 엔진, 직군 DB

### 4단계: 학습자 맞춤형 추천 시스템 개발 (12~15개월, 2차년도)

**13개월차** (2027.4월)
- 학습자 수준 진단 시스템 및 다차원 능력 프로파일 생성 알고리즘

**14개월차** (2027.5월)
- 맞춤형 문제 추천 알고리즘 (약점 보완, 적정 난이도 자동 조정)
- 학습 경로 최적화 알고리즘

**15개월차** (2027.6월)
- 동형 문제 대량 생성 시스템 (Paraphrasing 기반)
- 문제 풀 관리 시스템 구축
- **산출물**: 맞춤형 추천 알고리즘, 동형 문제 생성 엔진

### 5단계: 통합 검증 및 최적화 (15~18개월, 2차년도)

**16개월차** (2027.7월)
- 통합 평가 및 학습 플랫폼 구축 (React Native 앱 + 컨테이너 기반 API)
- 평가 모드 + 학습 모드 통합, Cloud 배포 및 AI Pipeline 자동화
- **장기 학습 효과 검증 시작**: 실험군/대조군 3개월 프로그램 시작

**17개월차** (2027.8월)
- 베타 테스터 100명 모집 및 파일럿 기업 3곳 실전 테스트
- 단기 학습 효과 검증 (4주간 맞춤형 학습 후 능력 향상 측정)

**18개월차** (2027.9월, 2차년도 종료)
- 장기 학습 효과 최종 평가 (3개월 완료 후 표준화 평가)
- 한국어교육 전문가 최종 평가 및 자문
- 성능 최적화 (속도, 비용, 확장성) 및 최종 보고서 작성
- **산출물**: 통합 플랫폼, 장기 학습 효과 검증 보고서, 전문가 평가 보고서, 최종 연구 보고서
