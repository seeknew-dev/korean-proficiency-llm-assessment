# 1. 기술개발 목표

## 1.1 연구개발 필요성 및 배경

현재 외국인 근로자 채용을 위한 한국어 능력 평가는 EPS-TOPIK(고용허가제 한국어능력시험)에 의존하고 있으나, 다음과 같은 근본적인 한계가 있습니다:

1. **직군별 언어 능력의 세밀한 측정 불가**:
   - EPS-TOPIK은 Volume 1(일상생활)과 Volume 2(직장생활)로 구성되어 있으나, Volume 2는 모든 직군의 내용을 혼합하여 출제
   - 숙박 서비스 지망생도 건축·기계 관련 용어를, 건설 지망생도 서비스·조리 관련 용어를 학습해야 함
   - **총점만 제공되므로** 기업은 지원자가 "자기 직군 문제를 맞췄는지" vs "다른 직군 문제를 맞췄는지" 알 수 없음
   - 결과적으로 **특정 직군에 필요한 실제 언어 능력을 정확히 측정할 수 없음**

2. **학습자의 비효율적 학습 부담**:
   - 모든 지원자가 자신의 희망 직군과 무관한 모든 산업 분야의 전문 용어를 학습해야 함
   - 예: 숙박 서비스 지망생이 "비계", "용접", "선반 가공" 등 기계/건설 용어를 암기
   - 예: 건축 시공 지망생이 "객실 정리", "체크인", "룸서비스" 등 서비스 용어를 암기
   - 실제 업무에서는 사용하지 않을 내용을 학습하는 시간적, 경제적 비효율성

3. **기업의 채용 미스매치 및 추가 비용 발생**:
   - 높은 EPS-TOPIK 총점을 받았더라도, 그 점수가 "자사 직군 관련 문제"에서 나온 것인지 확인 불가
   - 예: 건설사가 고득점자를 채용했으나, 실제로는 건설 용어는 틀리고 서비스 용어로 점수를 받은 경우
   - 기업은 채용 후 별도의 직군별 언어 교육을 진행해야 하는 추가 비용 발생
   - **고용허가제의 본래 취지인 "업무에 적합한 인력 선발"이 제대로 이루어지지 않음**

4. **낮은 변별력**: 단순 암기 학습으로 높은 점수를 받을 수 있어 실제 언어 능력 측정 한계

5. **제작 비용**: 고품질 문제 제작에 많은 시간과 인력 소요

본 과제는 LLM 기술을 활용하여 이러한 문제를 해결하는 차세대 한국어 능력 평가 시스템을 개발합니다:

**핵심 개선 방향**:
- **공통 영역**: 일상생활 한국어는 모든 지원자가 공통으로 평가 (EPS-TOPIK Volume 1과 동일)
- **직군별 영역**: 희망 직군에 맞춘 전문 용어 및 상황별 표현 평가 (선택적 평가)
- **세분화된 점수 제공**: "일상생활 점수" + "건축 시공 점수" 또는 "숙박 서비스 점수" 등 직군별 점수 분리 제공

이를 통해:
- **학습자**: 자신의 희망 직군에 필요한 언어만 집중적으로 학습 가능
- **기업**: 지원자의 "자사 직군 관련 언어 능력"을 정확히 파악하여 적합 인력 선발
- **평가 효율성**: 불필요한 학습 부담 제거 및 채용 미스매치 최소화

## 1.2 최종 개발 목표

**본 과제의 궁극적 목표**:

**"실수요자(국내 기업)에 맞춘 외국인 근로자의 한국어 능력 정확 평가 및 맞춤형 학습 지원을 통한 언어 능력 향상"**

현행 EPS-TOPIK은 모든 직군을 혼합 출제하여 총점만 제공하므로, 기업이 지원자의 "자사 직군 관련 언어 능력"을 정확히 파악할 수 없습니다. 본 과제는 LLM 기술을 활용하여 **직군별로 분리된 평가**를 제공함으로써, 기업이 자사 업무에 필요한 언어 능력을 정확히 측정하고 적합한 인력을 선발할 수 있도록 지원합니다.

나아가, 개발된 평가 시스템을 기반으로 **학습자 맞춤형 문제 제공**을 통해 외국인 근로자의 한국어 능력을 체계적으로 향상시킵니다:

**선순환 구조**:
1. **평가**: 직군별 세분화된 평가를 통해 학습자의 현재 수준 정확히 진단
2. **분석**: 다차원 난이도 측정 시스템으로 학습자의 강점/약점 영역 파악
3. **학습**: 학습자 수준에 맞는 문제를 자동 구성하여 제공 (약점 보완 + 단계적 난이도 상승)
4. **향상**: 지속적 학습을 통한 한국어 능력 향상 → 더 높은 수준의 평가 → 반복

**핵심 접근 방법**:
- 공통 영역(일상생활) + 직군별 영역(전문 용어) 분리 평가
- 실제 수험자 데이터 기반 문제 난이도 및 변별력 정밀 측정
- "일상생활 점수" + "직군별 점수" 세분화 제공
- **학습자 수준별 맞춤형 문제 세트 자동 구성** (평가 데이터 기반)

---

**세부 개발 목표**:

### 핵심 목표 1: 다차원 난이도 측정 시스템

- 문제의 난이도를 1차원 점수가 아닌 **다차원 벡터**로 측정
  - 어휘 난이도, 문법 복잡도, 독해 수준, 논리적 사고력, 실용적 기능 등
- 실제 수험자 응답 데이터 기반 **난이도 예측 정확도 90% 이상** 달성
- IRT(Item Response Theory) 기반 문항 변별도, 난이도 자동 계산

### 핵심 목표 2: 직군별 전문 언어 평가 시스템

- **공통 영역**: 일상생활 한국어 평가 (쇼핑, 병원, 은행, 교통 등 - 모든 직군 공통)
- **직군별 영역**: 각 직군에 특화된 전문 용어 및 상황별 표현 평가
  - 건축 시공: 안전 용어(비계, 추락 방지, 안전모), 공구명, 작업 지시 이해
  - 기계 조립: 공정 용어(선반, 용접, 조립), 작업 지시, 불량 보고
  - 음식 조리: 조리 기법, 위생 관리, 주문 처리, 식재료 명칭
  - 숙박 서비스: 객실 관리, 체크인/아웃, 고객 응대 표현
- **세분화된 평가 결과 제공**: "일상생활 점수" + "직군별 점수" 분리 산출
- 지정된 직군과 난이도에 맞는 **문제 세트 자동 생성**

### 핵심 목표 3: 학습자 맞춤형 추천 시스템

- 학습자의 **약점 영역 자동 진단** (어휘, 문법, 독해, 직군별 전문 용어 등)
- **적정 난이도 문제 자동 추천** (정답률 70~80% 수준 유지)
- **동형 문제 대량 생성 알고리즘** 구현 (반복 학습 지원)
  - 동일 난이도/변별력을 가진 서로 다른 문제 생성
  - 동형 문제 간 난이도 편차 **±5% 이내** 보장
